Lean Tools Toolkit

A snapshot of the lean manufacturing instruments I put into practice during my degree â€” from classic industrial engineering studies to collaborative Kaizen events that drive measurable results.

Yamazumi Chart

Stacked bars that instantly show which station is overloaded.

  • Each block = one task
  • Takt line highlights over/under
  • Great for quick rebalance talks

Time Study Sheet

Times every step so you know the true cycle time, not a guess.

  • Element-by-element timing table
  • Average and variation shown
  • Calls out gaps vs target time

Takt Time Calculation Sheet

Converts customer demand into the pace the line must run.

  • Uses net available minutes
  • Outputs takt + pitch in seconds
  • Flags when output slips behind

Standardized Work Combination Sheet

One line graph that shows work time, machine time, and waiting.

  • Instant overlap view
  • Highlights idle time
  • Helps right-size headcount

Standardized Work Chart

Simple map of the operator path, machine order, and SWIP limits.

  • Numbered route arrows
  • Safety and quality callouts
  • Visual SWIP caps on layout

Motion Analysis Sheet

Human motion wheel that captures reach, walk, search, and turn counts.

  • Logs motions by category
  • Flags fatigue or strain points
  • Guides workstation tweaks

OEE Sheet

One-page scorecard for availability, performance, and quality losses.

  • Downtime log + reason code
  • Performance vs ideal rate
  • Scrap / rework summary

SMED / Changeover Analysis Sheet

Breaks setup into internal vs external steps to slash changeover time.

  • Setup task checklist
  • Before/after time bar
  • Parallel work action plan

Andon / Abnormality Tracking Sheet

Quick log for every stop-and-notify event, defect, or operator alert.

  • Trigger + duration noted
  • Defect code and containment
  • Feeds daily tier boards

Poka-Yoke Sheet

Lists the failure mode and the simple device that prevents it.

  • Defect, cause, detection
  • Warning vs shut-down control
  • Verification / test noted

DFMEA Worksheet

Ranks design risk by severity, occurrence, and detection in one table.

  • Severity / occurrence / detection
  • Auto RPN and priority
  • Owner + due date column

Kaizen Proposal Sheet

One-page improvement story from current state through measured results.

  • Current vs target snapshot
  • Root cause + countermeasures
  • Before / after metric box
